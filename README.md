# CadastraPro

# Sistema de Cadastro de Clientes🚀

Sistema para cadastro, consulta, edição e exclusão de clientes em um comércio de varejo, facilitando o controle de informações e agilizando o trabalho da equipe de venda

# 🔌Como fazer funcionar na sua máquina:

-Instale Python 3.12;  
-Instale a biblioteca Tkinter;  
-Abra o terminal na pasta do projeto;  
-Execute o arquivo principal do sistema:  

# 📋Pré-requisitos do sistema:

Computador com Python instalado   
Sistema operacional: Windows  
Biblioteca Tkinter  
Banco de dados SQLite
